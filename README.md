# QSkill Login Form

## Description
A responsive Login Form developed using HTML, CSS, and JavaScript.

## Features

- Responsive Design
- Email Validation
- Password Validation
- Show/Hide Password
- Remember Me
- Forgot Password
- Alerts for Login Success and Errors

## Technologies Used

- HTML5
- CSS3
- JavaScript

## How to Run

1. Download the project.
2. Open in VS Code.
3. Install the Live Server extension.
4. Right-click `index.html`.
5. Select **Open with Live Server**.
